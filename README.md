# BookTalker

> 독서 후기 기록 및 공유 서비스

## 프로젝트 목표
- 나만의 서비스를 만드는 것에 대한 도전
- 확장 가능한 아키텍처 설계 경험

## 기술 스택
### Backend
- Spring Boot 3.x, PostgreSQL

### Frontend
- Vue3, PWA

### Infra
- Oracle Cloud Infrastructure(OCI)
- Nginx (리버스 프록시)
- Docker (개발 환경)

## 주요 기능
1. OAuth2 소셜 로그인 (Naver)
2. 책 검색 및 리뷰 작성
