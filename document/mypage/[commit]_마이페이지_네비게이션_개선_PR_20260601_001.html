PR 요약

마이페이지 독후감 카드 UX 개선 및 반복적으로 발생하던 네비게이션 탭 활성화 버그 2건 수정.

포함 커밋 (3건)

FEAT 3fc2666 마이페이지 단일 독후감 카드에 본문 미리보기 추가

1회독 단일 카드의 책 제목 아래 빈 공간에 독후감 본문 60자 미리보기 표시.
백엔드 ReviewListResponse에 content 필드 추가. 2회독 이상 카드는 기존 회독 배지 유지.

FIX a0dbe6a BookReviewsView 진입 시 홈 탭 활성화 안 되는 버그 수정

book-reviews 라우트에 meta: { activeNav: '/' } 추가.
review-detail, my-book-detail과 동일 패턴으로 통일.

FIX 8c5b915 로그아웃 시 nav 즉시 갱신 및 홈으로 리다이렉트

isLoggedIn을 useAuth.js 싱글턴 composable로 분리하여 AppHeader·useNav가 공유.
로그아웃 API 성공 즉시 isLoggedIn.value = false 반영 → 새로고침 없이 nav 갱신.
로그아웃 후 리다이렉트: login → home.
