주요 결정 사항

dev 대신 local 프로파일 사용
요구사항에는 dev로 명시됐으나, 프로젝트 기존 설정이 application-local.yml을 사용 중. dev를 새로 추가하면 혼란이 생기므로 local로 맞춤. 요청 시 dev 프로파일로 전환 가능.
프로퍼티 경로: logging.discord → app.discord
logging.*은 Spring Boot가 자체 관리하는 네임스페이스라 커스텀 키를 넣으면 IDE 경고 발생. app.discord.webhook-url로 변경하고 additional-spring-configuration-metadata.json에 등록해 경고 제거.
Discord 메시지 길이 제한 미적용
Discord Webhook 메시지는 2,000자 제한이 있음. 현재 구현은 스택트레이스를 포함하지 않고 getFormattedMessage()만 전송하므로 대부분 문제없음. 단, 메시지 자체가 긴 경우를 고려한다면 truncate 로직 추가 필요. (이번 범위 제외)
