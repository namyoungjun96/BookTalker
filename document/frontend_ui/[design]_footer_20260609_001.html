Footer 추가 설계서

[design]  작성일: 2026-06-09  |  대상: book_talker_frontend  |  작성: Claude

1. 목표 및 요구사항

  • 사이트 하단에 서비스 피드백 이메일(fuswns96@naver.com)을 노출하는 Footer 추가
  • 콘텐츠가 짧을 때도 스크롤 없이 한 화면에 Footer가 보여야 함
  • 콘텐츠가 길어지면 Footer는 아래로 자연스럽게 밀려남 (고정 오버레이 아님)
  • 로그인 페이지(route.name === 'login')에는 Footer 미노출
  • 기존 디자인 시스템 유지 (흰 배경, #e5e7eb 구분선, max-width: 720px)

2. 현재 구조와 문제점

현재 (문제)

각 View가 min-height: 100vh를 직접 보유
→ Footer를 단순 추가 시 스크롤해야 보임

목표

App.vue가 Flex 레이아웃으로 전체 높이를 관리
→ 콘텐츠가 짧아도 Footer가 하단에 위치

현재 각 View(HomeView, MyPageView 등)가 .app-container { min-height: 100vh }를 갖고 있어, 단순히 Footer를 추가하면 총 높이가 100vh + Footer 높이가 되어 스크롤이 발생합니다.

3. 설계안 — Sticky Footer 패턴

레이아웃 구조

AppHeader (sticky)
<main flex:1> → RouterView (각 View)
AppFooter

Flex column, min-height: 100vh

동작 원리

  • App.vue 래퍼: display:flex; flex-direction:column; min-height:100vh
  • <main>(RouterView 감싸기): flex:1 → 남은 공간을 모두 차지
  • 콘텐츠가 짧으면 → main이 늘어나 Footer를 하단에 고정
  • 콘텐츠가 길면 → main이 콘텐츠 높이만큼 늘어나 Footer가 자연스럽게 아래로
  • 각 View의 min-height: 100vh → min-height: 100%로 교체
    (부모 flex가 높이를 결정하므로 100vh 불필요)

App.vue 변경안

<template>
  <div class="app-layout">
    <AppHeader v-if="route.name !== 'login'" />
    <main class="app-main">
      <RouterView />
    </main>
    <AppFooter v-if="route.name !== 'login'" />
  </div>
</template>

<style>
.app-layout {
  display: flex;
  flex-direction: column;
  min-height: 100vh;
}
.app-main {
  flex: 1;
  display: flex;
  flex-direction: column;
  background: #f9fafb;   /* 배경색 통합 */
}
</style>

각 View 변경 포인트

App.vue가 배경(#f9fafb)과 최소 높이를 담당하므로, 각 View에서 중복 제거:

/* 변경 전 (각 View) */
.app-container {
  min-height: 100vh;
  background-color: #f9fafb;   /* 삭제 */
}

/* 변경 후 */
.app-container {
  min-height: 100%;   /* flex 부모 기준으로 채움 */
  /* background 제거 — app-main에서 처리 */
}

6. 트레이드오프

항목설명
장점 콘텐츠 길이에 상관없이 Footer가 항상 자연스러운 위치에 노출됨.
고정(fixed) Footer와 달리 콘텐츠 영역을 침범하지 않음.
변경 범위 View 파일 7개를 수정해야 함. 하지만 각 파일당 변경은 2줄(CSS 2줄 삭제/수정) 수준으로 단순.
주의 View별로 background 색상이 다를 경우 app-main 배경과 충돌 가능.
현재 모든 View가 동일하게 #f9fafb를 사용하므로 문제 없음.
