변경 내용

FIX #1 nextReadingCount '?' 고정 문제

원인: ReviewCreateView에서 nextReadingCount = ref('?') 로 하드코딩.

해결: MyPageView에서 readingCount를 query param으로 전달 → computed(() => Number(route.query.readingCount || 1) + 1)으로 계산.

FIX #2 독후감 작성 화면 nav 활성화 안 되는 문제

원인: review-create 라우트에 meta.activeNav 없어서 어떤 탭도 활성화 안 됨.

해결: useNav.js에 review-create 분기 추가 — mode=next-reading이면 마이페이지, 아니면 검색 탭 활성화.

FEAT 마이페이지 리뷰 구조 개편

기존: 모든 리뷰를 flat 목록으로 표시 (같은 책의 1·2·3회독이 각각 카드로 나열).
